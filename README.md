# Retail Sales Analysis Using SQL

# Project Overview 
This project focuses on analyzing retail sales transactions using SQL. Various business qusetions were sloved to gain insights into sales performance, customer demographics, product categories, and purchasing behavior. The project demonstrates practical SQL skills including filtering, aggregation, grouping, date manipulation, and business reporting.

## Project Objectives 
- Analyze retail sales performance
- Identify top-selling product categories
- Calculate total revenue and average sales
- Understand customer purchasing behavior
- Generate business insights using SQL queries
